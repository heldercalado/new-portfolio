Group Project

Live Demo Git Repo Page Analitics

For this Project we've came up with the idea of an app that would help people travelling to another city find a few things that are not immediatly easy to find without knpowing someone local to that place.

The information we wanted to add to the app:

Pictures of the City:

For this section we've used the google api (places + geocode) , in order to find the city name , latitude and longitude so we could that information to trigger the other apis used in the project

Once we've managed to find this information , we've used the top 10 found pictures from google places to populate a Bootstrap carousel component with the pictures.

Events:

For this section we've used the Eventbrite api with the response information returned from google geocode (Latitude , longitude)

This function also takes a ElementId argument that will serve as target for us to parse all the relevant information to that section. (used jquery to handle the parsing)

Afte all the information is parsed , at the end of each event we then created two buttons one that will take the user to the respective eventbrite main page , and other that will take the user to google maps to get directions on how to get there

Restaurants:

For this section we've used the Zomato api with the response information returned from google geocode (Latitude , longitude)

This function also takes a ElementId argument that will serve as target for us to parse all the relevant information to that section. (used jquery to handle the parsing)

After all the information is parsed , at the end of each restaurant we then created two buttons one that will take the user to the respective Zomato main page , and other that will take the user to google maps to get directions on how to get there

National Parks:

For this section we've used the NPS api with the response information returned from google geocode (city , state)

This function also takes a ElementId argument that will serve as target for us to parse all the relevant information to that section. (used jquery to handle the parsing)

  • Firebase
